# ft_printf

## Description

The ft_printf project consists of recreating the behavior of the standard C printf function.
The goal of this project is to better understand variadic functions, formatted output, and low-level string handling in C.

## Resources

### Supported conversions

- %c : character
- %s : string
- %p : pointer address
- %d / %i : signed integers
- %u : unsigned integers
- %x / %X : hexadecimal numbers
- %% : percent sign
